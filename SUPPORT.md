# Support

## Where To Ask What

### Bug reports

Open a GitHub issue with the bug report form.

### Feature requests

Open a GitHub issue with the feature request form.

### Documentation problems

Open a docs issue or submit a pull request.

### Security issues

Do not open a public issue. Follow [SECURITY.md](SECURITY.md).

### Demo or project questions

Start with the documentation:

- [README.md](README.md)
- [docs/getting-started.md](docs/getting-started.md)
- [docs/architecture.md](docs/architecture.md)

If the repository is published with a maintainer contact surface, use that private contact path for project-specific questions.

## Support Scope

This is a portfolio/demo repository. Support is best-effort and focused on:

- clarifying setup
- fixing repository issues
- documenting the active Cloudflare stack

It is not a commercial support channel for a live store, because the store itself is a concept project.
