# Usage Notice

`NIKITKA PREMIUM` is a public portfolio repository, not a reusable starter kit.

## Allowed

- viewing the source for evaluation
- reading the documentation
- discussing the implementation

## Not Allowed Without Written Permission

- redeploying this project as your own bot or storefront
- reusing the `NIKITKA PREMIUM` name, branding, or presentation
- copying the repository into a commercial or portfolio project
- mirroring the live bot identity or public demo as if it were your own work

## Why This Notice Exists

The repository is public so people can review the architecture and implementation quality. It is not published as open-source software for reuse.
