# Mini App UI Dropzone

Складывай все сгенерированные изображения интерфейса сюда:

- `assets/miniapp-ui/inbox/`

Сюда можно класть:

- фоны
- значки
- орнаменты
- текстуры
- декоративные ленты
- пустые состояния
- кнопочные акценты

После загрузки я сам:

- разберу файлы
- переименую
- разложу по нужным путям
- подключу в Mini App
