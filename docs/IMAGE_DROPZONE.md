# Image Dropzone

Класть исходные изображения сюда:

- `assets/products/inbox/`

После этого я сам:

- просмотрю изображения
- разберу, какой товар к какому относится
- переименую в нормальный формат
- подготовлю итоговые URL для бота
- обновлю каталог

Целевой нейминг для итоговых файлов:

- `silk-touch-tee-01.jpg`
- `monogram-tee-01.jpg`
- `tailored-hoodie-01.jpg`
- `studio-zip-hoodie-01.jpg`
- `raw-indigo-jeans-01.jpg`
- `modern-straight-jeans-01.jpg`

Если будут дополнительные ракурсы:

- `silk-touch-tee-02.jpg`
- `tailored-hoodie-02.jpg`

Отобранные и утвержденные изображения будут переноситься в:

- `assets/products/approved/`

Промпты для генерации лежат в:

- [docs/AI_PRODUCT_IMAGE_PLAN.md](/m:/Projects/Bot/test1/docs/AI_PRODUCT_IMAGE_PLAN.md)
